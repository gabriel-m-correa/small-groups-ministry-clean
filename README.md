# Small Groups Ministry App

Lightweight Next.js + Supabase MVP for managing small-groups intake, follow-up, and leadership development.

## Included

- Warm homepage
- Group interest form
- Leadership interest form
- Admin dashboard
- People pipeline with status tracking
- Groups directory
- Tasks and reminders
- Simple Supabase SQL schema

## Setup

1. Install Node.js.
2. Copy `.env.example` to `.env.local`.
3. Create a Supabase project.
4. Run the SQL in [supabase/schema.sql](C:/Users/GMC5F5/OneDrive%20-%20AdventHealth/Documents/New%20project/supabase/schema.sql).
5. Create at least one staff user in Supabase Auth.
6. Install dependencies and start the app.
